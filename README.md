# Filicori QR Studio for GitHub Pages

Acest folder este pregatit pentru publicare pe GitHub Pages.

## Varianta recomandata

1. Creezi un repository nou pe GitHub, de exemplu `filicori-qr-studio`.
2. Uploadezi continutul acestui folder in repository.
3. In GitHub mergi la `Settings` -> `Pages`.
4. La `Build and deployment` alegi:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/docs`
5. Salvezi.

GitHub va publica aplicatia pe un URL de forma:

`https://utilizatorul-tau.github.io/filicori-qr-studio/`

## Ce publici

Publici doar generatorul QR.

QR-ul creat de el poate continua sa trimita catre:

`https://filicorizecchini.ro/`
